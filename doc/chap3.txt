  [1X3.1 [33X[0;0YProfiling and special elements[133X[101X
  
  [33X[0;0YThe  FFCSA  package  contains  a  set  of profiling methods that are closely
  linked to the search for [13Xspecial[113X elements and polynomials. The [5XFFCSA[105X package
  [13Xprofiles[113X  are always a (set of) Hamming weight(s): a theoretical estimate of
  area, as was already noted for the [2XWeightMatrix[102X ([14X1.2-4[114X) method, or delay, as
  was  noted for the [2XWeightMatrixMaxRow[102X ([14X1.2-4[114X). Motivation example [14X1.1-1[114X is a
  search  for a small complexity normal basis. The particular normal basis has
  an  impact  on complexity of the arithmetic performed with field elements in
  normal  basis representation. Normal bases can be evaluated using [13Xcomplexity[113X
  [22XC_T[122X  defined  as  the  number  of nonzero elements in [13Xmultiplication table[113X [22XT[122X
  [MP13,  (5.3.1)].  Multiplication  table T is an [22X(mÃ— m)[122X matrix [22XT=[t_ij][122X over
  [22XF_q[122X,  defined  for  a particular normal element [22XÎ±âˆˆ F_2^m[122X, in such a way that
  the coefficients [22Xt_ij[122X satisfy[133X
  
  
  [24X[33X[0;6Y\alpha\cdot\alpha^{q^i}=\sum_{j=0}^{m-1}  t_{ij}\alpha^{q^j}  \quad {\rm for
  }\quad 0\leq i\leq m-1.[133X
  
  [124X
  
  [33X[0;0YThe complexity is bounded by [22X2m-1â‰¤ C_Tâ‰¤ m^2-m+1[122X, and when [22XC_T=2m-1[122X the basis
  is said to be \emph{optimal normal basis}, abbreviated ONB [MP13, (5.2.5)].[133X
  
  [1X3.1-1 IsONBI[101X

  [1X3.1-6 ProfileNBtoBTransitionMat[101X
  
  [33X[1;0Y[29X[2XProfileNBtoBTransitionMat[102X( [3XB[103X ) [32X method[133X

  [33X[0;0Y[10XProfileNBtoBTransitionMat[110X   finds  all  normal  elements  of  the  field  [3XF[103X,
  generates their normal bases [10XNB[110X, and computes their profile [10X[d1, A1, d2, A2,
  e][110X for the given basis [3XB[103X. Value [10Xe[110X is the exponent of the normal element, [10Xd1,
  A1[110X are the delay and area of the [22XNBâ†’ B[122X transition matrix, and [10Xd2, A2[110X are the
  delay   and   area  of  the  inverse  transition  matrix  [22XB  â†’  NB[122X.  Methods
  [2XTransitionMatrix[102X  ([14X1.3-2[114X)  and  [2XWeightMatrixBoth[102X ([14X1.2-4[114X) are used to compute
  the  profile.  [10XFindSmallestAreaNBtoBProfile[110X  returns  the  profile  with the
