4.1 Auxiliary functions

4.1-1 ChooseFieldElm
‣ ChooseFieldElm( F )( function )
‣ ChooseFieldElmdownto( F )( function )

In order to generate the expressions for hardware implementations, the GAP variables $a_i, b_j$ need to be created for symbollic computatioin. Method ChooseFieldElms prepares vectors \(avec=\left[a_0, \dots , a_{m-1}\right]\), \(bvec=\left[b_0,\dots,b_{m-1}\right]\), and \(dvec=\left[d_0,\dots,d_{2m-2}\right]\), with default direction {\tt To}, where \(m=[F:K]\) is the degree of extension. Vector \(dvec\) is needed for the two-step classic multiplication.

Method ChooseFieldElmsDownto creates vectors \(avec=\left[a_{m-1}, \dots , a_0\right]\), \(bvec=\left[b_{m-1}, \dots , b_0\right]\), and \(dvec=\left[d_{2m-2},\dots,d_0\right]\), with default direction {\tt To}, Note that \(a_i, b_j, d_k\) are not coefficients, but GAP variables to allow symbolic computation.
